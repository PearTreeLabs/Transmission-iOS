# Transmission-iOS

## Description

A [Transmission](http://www.transmissionbt.com) client for iOS.

## Usage

- Load submodules

        git submodule --init --recursive update

- Build and run the project using Xcode.

## Dependencies

- AFNetworking
- SSKeychain
