# Transmission-iOS

## Description

A [Transmission](http://www.transmissionbt.com) client for iOS.

## Usage

- Load submodules

        git submodule update --init --recursive

- Build and run the project using Xcode.

## Dependencies

- AFNetworking
- BPFoundation
